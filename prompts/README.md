# 🛡️ Prompt Management (Private)

To protect the intellectual property and the specific "secret sauce" of your anchoring prompts, this folder is configured to ignore `.md` files in Git (except for this README).

## How to use:
1. Create your prompts locally as `.md` files (e.g., `prompts/my_anchor_prompt.md`).
2. Git will **not** track these files due to the `.gitignore` configuration.
3. You can safely develop your methodology without exposing the exact strings to the public repository.

## Template structure (for local use)
You can follow this logic without committing the content:
- **Photography Tokens**: [Your private camera settings]
- **Anatomical Markers**: [Your unique identification tags]
- **Weighting**: [Your specific numerical values]
